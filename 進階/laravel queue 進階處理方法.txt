Horizon 支援的 queue driver

Redis ✅ 必須

Database / Beanstalkd / SQS / sync ❌ 不支援

原因

Horizon 需要：

快速的鎖機制 → 只有 Redis 能原子操作，支援 WithoutOverlapping 或類似行為

即時監控 → 儀表板會讀 Redis 來顯示 queue 狀態、job metrics

Supervisor / process 協調 → 多個 worker 同步狀態只能靠 Redis
